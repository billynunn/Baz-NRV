# Baz-NRV

Data Collection:
- UK Police Data (https://data.police.uk/data/) for crime rates in Basildon.
- Open Geography Portal (https://geoportal.statistics.gov.uk) for LSOA 2021 boundaries.
- ONS Census Data (https://www.ons.gov.uk/census) for socio-economic variables such as housing tenure, ethnicity. Unemployment was also tested but not included in the report.
